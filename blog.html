What is the Document Object Model (DOM) ?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

Event delegation
  1. Concept: Instead of attaching event listeners directly to individual elements, you attach a single listener to a parent element.
  1. Mechanism: The listener checks the target property of the event to determine which child element was actually clicked/interacted with.
How to you select an element form The DOM

Event delegation in the DOM is a technique where you attach a single event listener to a parent element instead of attaching multiple listeners to each child element. This listener checks the target property of the event to determine which child element was actually clicked/interacted with.


Benefits:
  1. Improved Performance: Reduces the number of event listeners, leading to better performance, especially with many dynamically added elements.
  1. Code Maintainability: Easier to add/remove event handlers for new elements.
